# Math formulas
## Area
- Circle: S = πR²
```python
def area(r):
    return math.pi * r * r
```
- Rectangle: S = ab
```python
def area(a, b): 
    return a * b
```
- Square: S = a²
```python
def area(a):
    return a * a
```
- Triangle: S = a * h / 2
```python
def area(a, h): 
    return a * h / 2 
```

## Perimeter
- Circle: P = 2πR
```python
def perimeter(r):
    return 2 * math.pi * r
```
- Rectangle: P = 2a + 2b
```python
def perimeter(a, b): 
    return (a + b)*2
```
- Square: P = 4a
```python
def perimeter(a):
    return 4 * a
```
- Triangle: P = a + b + c
```python
def perimeter(a, b, c): 
    return a + b + c 
```
